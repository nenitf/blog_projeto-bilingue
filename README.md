# [Projeto bilíngue](http://b.neni.dev)

Textos com audio e tradução, para treinar tanto *reading* quanto *listening*

## Desenvolvimento

### Setup local

- Baixar [hugo](https://github.com/gohugoio/hugo/releases)
- Subir servidor com ``hugo server -D -w``
